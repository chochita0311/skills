# AI Assets

Reusable AI assets for project work, centered on portable Codex skills and shared agent-system materials.

## What Lives Here
- `skills/`: reusable skill packages with `SKILL.md` entry contracts, references, templates, and skill-local metadata
- `agents/`: reusable harness role contracts and shared execution templates
- `agents/policies/`: reusable harness governance and evaluation assets shared with the agent package


## How To Navigate
- Start with [AGENTS.md](AGENTS.md) for the operational repo map and working rules.
- Read [skills/skill-quality.md](skills/skill-quality.md) for the quality bar for reusable skills.
- Read [agents/README.md](agents/README.md) for the shared agent-role package.

- Use each skill package's `SKILL.md` as the entry contract for that skill.

## Repository Intent
- Keep assets portable across consuming repos.
- Prefer one clear owner for each durable rule.
- Strengthen existing contracts before adding parallel guidance.
